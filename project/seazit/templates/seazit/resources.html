Resources

NTP hosted a workshop to solicit global input on assays, data analysis strategies, and regulatory feedback for innovative screening tools to better prioritize compounds with potential to cause developmental and adult neurotoxicity. A summary of discussions that occurred during the NTP-hosted workshop is available (Adobe PDF ).

Data and Peer Reviewed Publications

All the participants of the workshop have graciously consented to their data being made publicly available to foster scientific discussions and further the field. The data are available in text and Excel file formats in a zip file, and can be downloaded via the NTP CEBS data page. This also includes the results of the standardized NTP analysis including normalization and BMC analysis.

To cite these data, please use the following citation:

National Toxicology Program. 2018. Data Release: Developmental NeuroToxicity Data Integration and Visualization Enabling Resource (DNT-DIVER). Last Updated: December 3, 2018. doi: https://academic.oup.com/toxsci/pages/ntp_collection.

Related peer-reviewed publications in the scientific literature are also available in the above link.
